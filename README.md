# TuckerAPP-code
The released code of TuckerAPP algorithm for traffic data imputation, mainly proposed in the paper 
"<a href="https://GongWenwuu.github.io/mypaper/TuckerAPP_ITS.pdf"> TuckerAPP: A novel spatiotemporal Tucker decomposition approach for traffic imputation </a>", which is accepted by the journal Signal Processing, 2025.
